How to join

We appreciate your help testing FluidCast. Follow these quick steps to join the TestFlight beta:

1. Install TestFlight

If you don't already have TestFlight, install it from the App Store on your iPhone.

2. Tap the Join Link

On your iPhone, tap the Join on TestFlight button above (or paste the link into Safari). This will open TestFlight and let you accept the beta invite.

3. Install and Test

Install the beta build from TestFlight. Please try core features and report issues to support@fluidcastapp.com with the build number and steps to reproduce.

Notes

TestFlight builds may expire. If the link stops working or the build is full, contact support and we'll add you to the next batch.

Request Pro Entitlement

If you’d like to receive Pro access (cloud sync, premium features), tap the Test Pro Features button above and fill in the Google Form. Provide the email address asociated with your account, and you will be granted 3 months of Pro once approved, (up to a day).
